Actividad 2: Color

Sistema de control de calidad de galletitas variadas.

A partir de fotos de galletitas Variedad (o similar) estimar cuántas hay de cada color. Las fotos pueden ser tomadas asumiendo fondo blanco (o controlado) y la cámara siempre a la misma altura.

Para resolverlo:
	1. ¿Qué sistema de color conviene usar para distinguir los diferentes colores?
	2. Establecer rangos para cada color.
	3. ¿Cuántos píxeles hay de cada color?
	4. ¿Cuántos píxeles ocupa cada galletita?
